## Deskripsi

Tugas ini adalah aplikasi manajemen tugas berbasis Laravel yang digunakan untuk mencatat tugas, mengubah tugas, menghapus tugas, menandai tugas selesai, melihat dashboard statistik, dan melihat laporan tugas berdasarkan rentang tanggal.

## Teknologi

* Laravel 13
* SQLite
* Bootstrap 5
* Git
* GitHub

## Fitur

* Menampilkan daftar tugas
* Menambah tugas
* Mengubah tugas
* Menghapus tugas
* Menandai tugas selesai
* Dashboard statistik
* Laporan tugas berdasarkan tanggal
* Testing Laravel

## Cara Menjalankan

Install dependency:

composer install

Generate key:

php artisan key:generate

Migrasi database:

php artisan migrate

Menjalankan aplikasi:

php artisan serve

Akses:

http://127.0.0.1:8000
